Important information

  • The IAR C-SPY ROM-monitor for 8051 is a modular monitor designed to work with a number of on-board/chip monitors. Refer to the manual for details on how to build the monitor.

Miscellaneous

  • The error message displayed when the code-size limit is reached in the Baseline or Kickstart edition incorrectly states the unit of the limit as Kbytes rather than bytes.

  • Downloading a non-UBROF file will not set PC to program start. You can work around this limitation by unchecking Run to main in the debugger options menu, download the application, and then set PC manually before execution is started.
