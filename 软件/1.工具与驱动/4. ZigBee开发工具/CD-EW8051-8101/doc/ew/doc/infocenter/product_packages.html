IAR Embedded Workbench for 8051 product packages

IAR Information Center for 8051 | Release notes | IAR Embedded Workbench product packages

Standard edition

The standard edition comes with all features.

Limited edition

The limited edition is a version of IAR Embedded Workbench without a debugger. Limitations:

  • The IAR C-SPY debugger is not available.
  • MISRA C is not available.

BaseLine edition

The BaseLine edition is a special low-priced version of IAR Embedded Workbench. Limitations:

  • The code size limit of the IAR Baseline C/C++ Compiler for 8051 is set to 8 Kbytes.
  • The IAR XLINK Linker will link a maximum of 8 Kbytes originating from C/C++ source code, but an unlimited amount of code originating from assembler code.
  • The IAR BaseLine C-SPY Simulator will read a maximum of 8 Kbytes originating from C/C++ source code, but an unlimited amount of code originating from assembler code.
  • MISRA C is not available.

KickStart edition

The KickStart edition is a special starter kit/evaluation version of IAR Embedded Workbench with limitations both in code size and in the service and support that is provided. Limitations:

  • The code size limit of the IAR KickStart C/C++ Compiler for 8051 is set to 4 Kbytes.
  • The IAR XLINK Linker will link a maximum of 4 Kbytes originating from C/C++ source code, but an unlimited amount of code originating from assembler code.
  • The IAR KickStart C-SPY Simulator will read a maximum of 4 Kbytes originating from C/C++ code, but an unlimited amount of code originating from assembler code.
  • MISRA C is not available.
  • The runtime library source code is not included.

Evaluation edition

The Evaluation edition is a time-limited evaluation version that you can use for 30 days. Limitations:

  • Will expire 30 days after the generation of the evaluation key.
  • MISRA C is not available.
  • The runtime library source code is not available.
